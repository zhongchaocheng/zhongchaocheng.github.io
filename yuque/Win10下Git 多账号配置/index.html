1.生成github.com第一个帐号对应的私钥公钥

执行命令 **ssh-keygen -t rsa -C email** 创建github对应的sshkey，命名为id_rsa_你的自定义名字，这里我取的是id_rsa_zcc，后面后面是输入提交修改时的密码，可以直接回车跳过。
